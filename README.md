# Hunnits

> Bash script to make it rain.

## Install

```
$ curl https://raw.githubusercontent.com/kjbrum/hunnits/master/hunnits > ~/bin/hunnits
$ chmod +x ~/bin/hunnits
```

## Usage

```
$ hunnits
```
